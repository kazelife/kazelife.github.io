写在前面

完全不推荐本文提到的智能锁，除非你技术力强到能让它闭嘴，或者对噪音完全不敏感。

虽然出门不用带钥匙很爽，但实在太吵了，对我来说弊远远大于了利，
尤其是频繁触发的一惊一乍的“（超吓人超揪心的警报声）！开锁失败！请输入开锁信息！”

怎么进入管理员模式？

请注意：此步骤为进行用户设置的前置条件

  1. 按一下#键唤醒屏幕，会提示“请输入开锁信息”
  2. 再长按#键，会提示“请输入管理员信息”
  3. 输入密码进入管理员模式

怎么添加用户？

  1. 进入管理员模式
  2. 按1“管理员设置”
  3. 按1“添加管理员”
  4. 依次输入2遍新密码或按5遍指纹
  5. 提示新管理员编号“00x添加成功”

怎么删除用户？

  1. 进入管理员模式
  2. 按1“管理员设置”
  3. 按2“删除管理员”
  4. 输入要删除的管理员编号“00x”

心态调整

屡次进入管理员模式失败，被重复无数次的“请输入管理员密码”和按九个键的疲惫操作给弄疯了！
想到了集成灶也是，太他妈烦了！！！

是我对这些屁用没有的话应激还是本就让人应激？
我并不是特例，来让集成灶闭嘴的小哥就说过很多人都有这个要求的：
大部分用户都他妈受不了这些电器的聒噪！

为什么我因为聒噪的电器过激了？

  1. 吵得头疼，一句话翻来覆去重复无数遍
  2. 把你当傻子
  3. 吵得你吓得你不敢再尝试

利用这一点，换你来设计，你要怎么做？

  1. 不说废话，什么狗屁“欢迎使用，请输入”什么的都滚远点！
  2. 不把用户当傻子，给用户点信心，用清晰的UI表示清楚就够了，用户看得懂的！
  3. 尝试成本一定要弄低，不要动不动就“已锁定”，动不动就开启了什么“常开模式”，
    这种摧毁性的字眼太打击了用户的探索心了！
