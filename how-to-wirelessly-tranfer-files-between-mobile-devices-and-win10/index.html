Win10怎么手机电脑互传文件？【无需下载】

我的需求

  1. iPad备忘录传Win10电脑上（下文将以iPad操作展开，并不代表iPhone和Android无法使用）
  2. 我受不了微信传输助手：特地打开麻烦、重复操作麻烦、后续清理麻烦
  3. 最好啥也不要依赖，不想为了传个文件：
    又开微信、又开qq、又开爱思助手、又开蓝牙、又开AirDrop、又插数据线的……
    我累不累啊！
  4. 别叫我下载东西，我烦；更别强制我注册登录，我更烦

方法1

想传输的设备都保持以下网站开启：

https://snapdrop.net

优点

  1. 满足了我的需求
  2. 操作简单
  3. 不限设备，有网，能开浏览器，就能互传

缺点

  1. 不稳定，有时候网站打不开
  2. 依赖第三方，网站挂了就又得重新找办法了

方法2

与其担忧snapdrop挂了的时候怎么办，不如找到更稳定的法子一劳永逸。

虽然很难找，我又啥也看不懂，好在还是让我找到了目前最完美的解决法！

这方法用到的是：Win10自带的Server Message Block (SMB) 网络传输协议

不用害怕，就算不知道smb是什么，不用逼自己搞清楚，也是完全不影响接下来的使用的！

电脑步骤

  1. 在电脑上新建一个文件夹作为共享文件夹
  2. 右键新建文件夹→属性
  3. 切到共享标签→高级共享
  4. 勾选共享此文件夹
  5. 权限→添加→输入everyone→确定（可自定义，本文以everyone为例）
  6. 勾选允许完全控制→确定
  7. 确定→关闭
  8. 查好电脑的内网IP地址备用（不知道怎么查？）

手机步骤

  1. 打开iPad自带的文件应用
  2. 点左上角，展开左侧栏
  3. 再点左侧栏的右上角横的三点···
  4. 选择连接服务器
  5. 输入smb://192.168.0.1
    数字部分改成上面第8步获取的你的IP地址
  6. 连接→连接身份选客人→下一步
  7. 能看到共享文件夹了

如果你也有转移备忘录到电脑的需求？

  1. 打开备忘录
  2. 点分享按钮→存储到“文件”
  3. 在左侧栏共享下选择你的IP地址
  4. 点你的共享文件夹→存储
  5. 就会自动出现在你电脑上的该共享文件夹内了
  6. 备注：只需手动选择一次，之后都会自动记住最近操作路径，不用每次选的

优点

  1. 满足了我的需求
  2. 配置几分钟，享受一辈子~
  3. 新增、删除、更改等操作，全是实时同步的
  4. snapdrop还得上传下载选保存路径呢，这方法极大地减少了鼠标操作量
  5. 无缝体验超棒，就像在电脑上操作文件一样自然舒适，很习惯，无需适应

    不禁想大喊：不需要用户适应的用户体验就是最高的用户体验！

缺点

  1. 配置有点麻烦
  2. 可能会卡在意外的报错上
