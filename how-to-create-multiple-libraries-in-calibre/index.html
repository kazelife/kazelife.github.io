怎么建立书库

以建立新书库kaze的第二个书库为例

  1. 主工具栏上默认有个按钮，显示的是你的自定义书库名
  2. 点那个按钮→切换 / 创建书库
  3. 开始填空
    新位置: D:\kaze-lib\kaze的第二个书库 # 改成自己想要的路径和书库名
    〇在新位置使用一个已经存在的书库 # 复制
    〇在新位置创建一个空白书库 # 新建
    〇将目前书库移动至新的位置 # 剪切
    □复制当前书库的结构 # 要不要把你的设置带过去，比如自定义栏目、搜索记录、栏目宽度、元数据处理规则等
  4. 确定

怎么建立虚拟书库

以标签栏目为例，筛选标签为标签1(tag1)的全部书籍

  1. 左上角虚拟书库
  2. 创建虚拟书库
  3. 虚拟书库名称: 标签1
  4. 搜索表达式: tag1
  5. 确定
