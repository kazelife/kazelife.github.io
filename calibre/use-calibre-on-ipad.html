How to use calibre on iPad

Steps as follows:

  1. Preferences
  2. Sharing over the net
  3. Start server
  4. Visit [your IP address]:[port] (default port: 8080)
    Any device on the same Wi-Fi can access to your library now.
    You can read it directly from browser like Safari or use apps like iBooks.
    I would personally recommend Safari for highlights sync.

Run server automatically when calibre starts:

  1. Preferences
  2. Sharing over the net
  3. Check Run server automatically when calibre starts
  4. Apply, OK, Close
  5. Restart calibre manually
