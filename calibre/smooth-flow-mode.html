How to stop chapter snapping in calibre flow mode

Caution: Step 5 is non-reversible!

If you still need seperated htmls, please make a backup first.

If you changed your mind at Step 4, you can revert it by Ctrl+Left.

Steps as follows:

  1. Right-click a book, Edit book
  2. Right-click Text, Select all Text files
  3. Right-click the chosen area, Merge selected text files
  4. Choose the master file, OK, Ctrl+S
  5. Close the calibre-ebook editor
