How to custom search behavior when you click author from Book details

Steps as follows:

  1. Preferences
  2. Look & feel
  3. Book details
  4. Clicking on author names should:
    • Search for the author on Goodreads
    • Search for the author on Amazon
    • Search for the author in your calibre library
    • Search for the author on Wikipedia
    • Search for the author on Google Books
    • Search for the book on Goodreads
    • Search for the book on Amazon
    • Search for the book on Google Books
    • Use a custom search URL
  5. Choose one you like or choose Use a custom search URL to set url manually

Custom search url templates:

    Attention: search url may varies from different search engines.
  1. e.g. Search author/book at Google
    • https://www.google.com/search?q={author}
    • https://www.google.com/search?q={title}
  2. e.g. Search author/book at Bing
    • https://www.bing.com/search?q={author}
    • https://www.bing.com/search?q={title}
  3. e.g. Search author/book at Douban (specific for book written in Chinese language)
    • https://book.douban.com/subject_search?search_text={author}
    • https://book.douban.com/subject_search?search_text={title}
  4. e.g. Search author/book at Baidu (default Chinese search engine, not recommended)
    • https://www.baidu.com/s?wd={author}
    • https://www.baidu.com/s?wd={title}
